## Size

Dimensions of the loader are tied to the global `scale` tokens. This ensures that the components with similar size settings remain consistent across the interface. `scale` tokens also provides more granular control over dimensional settings.

```css:scale.css showLineNumbers
--scale-width-<size>
--scale-height-<size>
```
