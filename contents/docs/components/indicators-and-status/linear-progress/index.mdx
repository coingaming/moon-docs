## Size

Dimensions of the linear progress are tied to the global `scale` tokens. This ensures that the components with similar size settings remain consistent across the interface. `scale` tokens also provides more granular control over dimensional settings.

```css:scale.css showLineNumbers
--scale-width-<size>
--scale-height-<size>
--scale-gap-<size>
--scale-radius-<size>
--scale-text-size-<size>
--scale-text-height-<size>
```
