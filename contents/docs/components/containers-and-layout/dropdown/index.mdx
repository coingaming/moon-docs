## Best practises

- Use dropdowns to present a list of options or selections.
- Keep dropdown options concise and easy to scan, avoid very long lists when possible.
- On mobile, prefer native select dropdowns to improve usability.
- Make sure to support keyboard navigation and screen readers for accessibility.

## Behaviour

Dropdowns expand vertically to reveal a list of selectable options when triggered. They overlay the content without navigating away.
Dropdowns close automatically when an option is selected or when clicked outside the dropdown area.

## Size

Dropdown width typically matches the trigger element but can expand wider to provide better readability.
Height of dropdown adjusts dynamically based on the number of options. Maximum height is set with `max-height` token to enable scrolling when the content overflows.
