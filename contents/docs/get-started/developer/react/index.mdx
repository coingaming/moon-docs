## Install moon-react package

```bash
# Using npm
npm install @moondesignsystem/react

# Using pnpm
pnpm install @moondesignsystem/react

# Using yarn
yarn add @moondesignsystem/react
```

## OR Install components using moon-react CLI

```bash
# Using npm
npx @moondesignsystem/react --all-components
```

To import components:

```tsx
import { Button } from "@moondesignsystem/react";

const App = () => <Button>Moon Button</Button>;

export default App;
```

OR To use locally installed components:

```tsx
import Button from "./components/Button";

const App = () => <Button>Moon Button</Button>;

export default App;
```
